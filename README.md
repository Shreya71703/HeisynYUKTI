# Codify — Empowering Tomorrow's Professionals, Today.

**Codify** provides a unified, all-in-one platform to guide students and professionals by offering essential academic and professional tools. It helps you move seamlessly from college to career with resources, connections, and innovative features.

---

## 🚀 Features

### 1. Guide Barter
A peer-to-peer platform using smart contracts for skill exchange and guided mentorship.

### 2. Build Next
AI-driven NLP tools for resume generation, resume analyzer, and custom GPT model.

### 3. Lounge
Encrypted, interest-based chat rooms powered by communication protocols.

### 4. Job Connect
AI-powered job-matching algorithms with automated application submissions and startup-friendly portals.

### 5. Inter(company's)view
Machine learning-based mock interviews with company-specific datasets and real-time feedback.

### 6. College Sync
Cloud-based access to semester-wise PYQs, notes, and video tutorials with seamless synchronization.

### 7. Alumni & Career Maps
Blockchain-verified alumni networks and AI-generated career road maps.

### 8. Code Wizard
Integrated coding IDE and competitive coding platforms.

### 9. Open Source & Freelance
GitHub API integration for tracking open-source contributions and freelance work, earning badges.

---

## ⭐ Our Stand Out Feature

### AI-Free Interview Hub
Codify offers an immersive AI-based audio interview covering every type of role and interview complexity.

## 📦 Getting Started

```bash
# Clone the repository
git clone https://github.com/Shreya71703/HeisynYUKTI.git

# Navigate to the project
cd HeisynYUKTI

# Install dependencies
npm install

# Start the development server
npm run dev
```
